Tipos de datos recopilados

Esta aplicación no recopila datos personales, ya sea directamente o a través de terceros. Por lo tanto, no requiere permisos de contactos, cámara, micrófono, cuentas de redes sociales, ubicación continua ni correos electrónicos.

Se utilizan exclusivamente identificadores únicos de dispositivos para la gestión de publicidad.

Modo y lugar de procesamiento de los Datos

  • Métodos de procesamiento: Al no recopilar datos personales, no se aplica un tratamiento específico de datos.
  • Lugar: No existe un lugar de almacenamiento centralizado al no haber captación de información personal.
  • Tiempo de retención: No existe un tiempo de almacenamiento de datos personales.
  • Fines del procesamiento: No existe un fin específico de procesamiento de datos personales.

Permisos de Redes Sociales

Esta aplicación no solicita ni requiere permisos de acceso a cuentas de Redes Sociales.

Anuncios

Esta aplicación contiene anuncios intersticiales integrados a través de la plataforma AdMob™.

Esta política está alojada en GitHub Pages para transparencia y acceso.
