1. Prácticas de seguridad

Todos los datos recopilados o procesados por esta aplicación se cifran mediante protocolos seguros (como HTTPS) durante su transmisión y uso.

2. Tipos de datos que recopila esta aplicación

  • 2.1. Ubicación aproximada: Se utiliza para fines estadísticos y de marketing a través de servicios de terceros (AdMob).
  • 2.2. Registro de errores e información de rendimiento: Datos técnicos para mejorar la estabilidad de la app.
  • 2.3. Datos del dispositivo: Identificadores técnicos para optimización y publicidad.
  • 2.4. Datos ingresados por el usuario: Los datos de salud (peso, estatura, semanas de gestación) se procesan localmente en el dispositivo para realizar los cálculos.

3. Métodos de procesamiento

El procesamiento de metadatos técnicos se realiza a través de terceros (Google SDKs) con fines de análisis y monetización. Los cálculos de IMC se ejecutan de forma local en el hardware del teléfono.

4. Retención de datos

Datos locales: Los registros de cálculos y perfiles creados dentro de la aplicación se almacenan únicamente en la base de datos local del dispositivo del usuario. Estos datos se conservan de forma indefinida mientras la aplicación esté instalada para permitir al usuario llevar un seguimiento de su historial.

Datos de terceros: Los datos analíticos y publicitarios se retienen según las políticas de privacidad de Google AdMob y Firebase.

5. Eliminación de datos y Derechos del usuario

El usuario tiene control total sobre su información y puede solicitar o ejecutar la eliminación de sus datos de la siguiente manera:

  • Eliminación manual: El usuario puede borrar registros individuales o el historial completo directamente desde la sección de configuración o historial dentro de la aplicación.
  • Eliminación total: Al desinstalar la aplicación o "Borrar datos de almacenamiento" desde los ajustes de Android, todos los datos locales se eliminan de forma permanente e irreversible.
  • Solicitudes por correo: Para cualquier consulta sobre la privacidad o para solicitar la eliminación de datos de soporte técnico, el usuario puede escribir a simplified.maed@gmail.com.

6. Redes Sociales

Esta aplicación no interactúa con las redes sociales vinculadas al usuario ni accede a sus credenciales de acceso.

Términos y Condiciones

  • Esta aplicación proporciona cálculos del Índice de Masa Corporal (IMC) según las directrices de la Organización Mundial de la Salud (OMS) y tablas de ganancia de peso gestacional.
  • La información proporcionada es solo para fines informativos y no sustituye el consejo médico profesional.
  • Consulte con un profesional de la salud antes de tomar cualquier decisión relacionada con su salud o peso.
  • El uso de esta aplicación es bajo su propia responsabilidad.
