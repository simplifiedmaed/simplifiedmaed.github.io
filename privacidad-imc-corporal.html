1. Prácticas de seguridad

Todos los datos recopilados por esta aplicación se cifran durante su uso.

2. Tipos de datos que recopila esta aplicación

  • 2.1. Ubicación aproximada: Se utiliza para fines estadísticos y de marketing.
  • 2.2. Registro de errores e información de rendimiento: Se utilizan para fines estadísticos y de marketing.
  • 2.3. Datos del dispositivo: Se utilizan para fines estadísticos y de marketing.
  • 2.4. Datos del usuario: La aplicación no recopila información personal del usuario.

3. Métodos de procesamiento

El procesamiento de datos como la ubicación aproximada, la información del registro de fallos, el rendimiento y los datos del dispositivo se realiza a través de un tercero con fines estadísticos y de marketing.

4. Lugar, Retención y Finalidad

El procesamiento, almacenamiento y tiempo de retención de los datos recopilados se realiza a través de un tercero con fines estadísticos y de marketing.

5. Redes Sociales

Esta aplicación no interactúa con las redes sociales vinculadas al usuario.

Términos y Condiciones

  • Esta aplicación proporciona cálculos del Índice de Masa Corporal (IMC) según las directrices de la Organización Mundial de la Salud (OMS).
  • La información proporcionada es solo para fines informativos y no sustituye el consejo médico profesional.
  • Consulte con un profesional de la salud antes de tomar cualquier decisión relacionada con su salud o peso.
  • El uso de esta aplicación es bajo su propia responsabilidad.
  • Los creadores de esta aplicación no se hacen responsables de ningún daño derivado del uso de esta aplicación.
