1. Prácticas de seguridad:

Todos los datos recopilados por esta aplicación se cifran durante su uso.

2. Tipos de datos que recopila esta aplicación:

2.1. Ubicación aproximada:

Los datos de ubicación aproximada se utilizan para fines estadísticos y de marketing.

2.2. Registro de errores e información de rendimiento:

Los datos de registro de fallos y rendimiento se utilizan para fines estadísticos y de marketing.

2.3. Datos del dispositivo que utiliza la aplicación:

Los datos del dispositivo que utiliza la aplicación se utilizan para fines estadísticos y de marketing.

2.4. Datos del usuario:

La aplicación no recopila información personal del usuario.

3. Métodos de procesamiento:

El procesamiento de datos como la ubicación aproximada, la información del registro de fallos, el rendimiento y los datos del dispositivo se realiza a través de un tercero con fines estadísticos y de marketing.

4. Lugar:

El procesamiento y almacenamiento de datos como la ubicación aproximada, la información del registro de fallos, el rendimiento y los datos del dispositivo se realiza a través de un tercero con fines estadísticos y de marketing.

5. Tiempo de retención:

El tiempo de retención de los datos recopilados, como la ubicación aproximada, el registro de fallos, el rendimiento y los datos del dispositivo, se realiza a través de un tercero con fines estadísticos y de marketing.

6. Finalidad del tratamiento:

El tratamiento de los datos recopilados, como la ubicación aproximada, el registro de fallos, el rendimiento y los datos del dispositivo, se realiza a través de un tercero con fines estadísticos y de marketing.

7. Permisos de redes sociales solicitados por esta aplicación:

Esta aplicación no interactúa con las redes sociales vinculadas al usuario.
