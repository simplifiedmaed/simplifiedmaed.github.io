1. Prácticas de seguridad

Todos los datos recopilados o procesados por esta aplicación se cifran mediante protocolos seguros (como HTTPS) durante su transmisión y uso. No vendemos ni compartimos información personal o de salud con terceros para fines ajenos al funcionamiento de la publicidad integrada.

2. Tipos de datos que recopila esta aplicación

En cumplimiento con las políticas de Google Play, declaramos la recopilación y uso de los siguientes datos:

  • 2.1. Información Personal: Recopilamos el nombre del usuario únicamente con fines de personalización de la interfaz. Este dato no se comparte con terceros.
  • 2.2. Ubicación aproximada: Se utiliza exclusivamente para fines de publicidad o marketing a través de proveedores externos como Google AdMob.
  • 2.3. Información de rendimiento: Recopilamos registros de fallas y diagnósticos de forma anónima para generar estadísticas y mejorar la estabilidad de la app.
  • 2.4. Dispositivo u otros IDs: Se recopilan identificadores técnicos para fines de publicidad, marketing y prevención de fraudes. Estos IDs se comparten con el SDK de Google AdMob.

3. Métodos de procesamiento

El procesamiento de metadatos técnicos y publicitarios se realiza a través de terceros (Google SDKs). Los cálculos de Energía Solar se ejecutan de forma local en el hardware del dispositivo del usuario para garantizar la máxima privacidad.

4. Retención de datos

Datos locales: El historial de cálculos y el nombre del proyecto de usuario se conservan de forma indefinida en la base de datos local del dispositivo mientras la aplicación esté instalada.

Datos de terceros: Los datos analíticos y de publicidad se retienen según las políticas de privacidad de Google AdMob y Firebase.

5. Eliminación de datos y Derechos del usuario

El usuario puede gestionar su información de la siguiente manera:

  • Eliminación manual: El usuario puede borrar registros individuales o el historial completo desde la configuración de la app.
  • Eliminación total: Al desinstalar la app o borrar el almacenamiento en los ajustes de Android, todos los datos locales se eliminan permanentemente.
  • Solicitudes: Para cualquier consulta, escriba a simplified.maed@gmail.com.

Términos y Condiciones

  • Esta aplicación proporciona cálculos para el sistema de energía solar, carga de electricidad y energía solar disponible en la zona del usuario.
  • La información proporcionada es solo para fines informativos y no sustituye el consejo profesional técnicop.
  • El uso de esta aplicación es bajo su propia responsabilidad.
